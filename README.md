# deqwl — VLESS Checked Subscription (оптимизировано для РФ)

**Ссылка для импорта:** https://raw.githubusercontent.com/dequar/deqwl/refs/heads/main/deray.txt

**Как импортировать:**
- v2rayNG / Nekobox / Hiddify → Добавить подписку → вставить ссылку выше
- v2rayN (Windows) → Subscription group → Add URL
- Обновляется автоматически

**Особенности:**
- Берёт самые качественные источники: подписки igareck для мобильных устройств - и объединяет в одну, для удобства и лучшей производительности
- Конфиги отсортированы (Россия сверху)
- Красивые названия
- Максимально высокий процент рабочих конфигов под белые списки РФ

## Запуск сборщика вручную

**Консольная версия:**
pip install requests
python collector.py
**GUI версия (с графическим интерфейсом):**
pip install requests
python collector_gui.py
> tkinter уже встроен в Python, дополнительно устанавливать не нужно.

**GUI позволяет:**
- Добавлять и удалять источники без редактирования кода
- Видеть лог проверки в реальном времени
- Запускать сборку кнопкой, не заходя в терминал
