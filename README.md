# deqwl — VLESS Checked Subscription (оптимизировано для РФ)

**Основная ссылка (рекомендуется):** https://raw.githubusercontent.com/dequar/deqwl/main/vless_checked.txt

**Для тех, кто хочет БОЛЬШЕ (не для телефонов!):** https://raw.githubusercontent.com/dequar/deqwl/main/vless_all.txt

**Как импортировать:**
- v2rayNG / Nekobox / Hiddify → Добавить подписку → вставить ссылку выше
- v2rayN (Windows) → Subscription group → Add URL
- Обновляется автоматически

**Особенности:**
- Берёт самые большие и качественные источники: zieng2 (vless.txt) + igareck WHITE-SNI-all + WHITE-CIDR-all (да, этот проект - всего лишь агрегатор, все данные взяты из открытых источников)
- Конфиги отсортированы (Россия сверху)
- Красивые названия (с флагами и айди в конфигах страны)
- Максимально высокий процент рабочих конфигов под белые списки РФ
